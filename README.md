## ▶️ Getting Started

### Prerequisites

- Node.js 18 or higher
- npm

### Installation

```bash
git clone <your-repo-url>
cd mpfstudio91
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.

### Build for Production

```bash
npm run build
```
