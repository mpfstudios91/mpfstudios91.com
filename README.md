# YamlTech Landing Page

A modern, responsive landing page for YamlTech - a professional IT solutions and digital services company. Built with vanilla HTML, CSS, and JavaScript for optimal performance and maintainability.

![mpfstudio91](https://cometakebucket2.s3.amazonaws.com/email_templates/yamltechFavicon.png)

## 🚀 Features

- **Fully Responsive Design** - Optimized for all devices (mobile, tablet, desktop)
- **Modern UI/UX** - Clean, professional design with smooth animations
- **SEO Optimized** - Comprehensive meta tags, semantic HTML, and accessibility features
- **Fast Loading** - Vanilla CSS/JS with no heavy frameworks
- **Interactive Elements** - Smooth scrolling, mobile menu, back-to-top button
- **Contact Form** - Integrated contact form with validation

## 🚦 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:

```bash
git clone https://github.com/chikamso-cmd/Yamltech.git
cd Yamltech
```

2. Install dependencies:

```bash
npm install
```

3. Start development server:

```bash
npm run dev
```

4. Open your browser to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The optimized files will be in the `dist/` directory.
